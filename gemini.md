# Gemini Project Context

This project is designed to facilitate the installation of OpenWebUI in an intranet environment.

## File Structure

- **`internet/download_packages.bat`**: This script is intended to be run on a machine with internet access to download all necessary packages for the OpenWebUI installation.
- **`intranet/install_openwebui.bat`**: This script is used on an intranet machine (without internet access) to install OpenWebUI using the packages downloaded by the `download_packages.bat` script.
- **`intranet/run_openwebui.bat`**: This script runs the installed OpenWebUI application.
- **`docs/`**: Contains documentation for the project.
- **`Claude.md` / `README.md`**: General documentation files.
